# Diagrama de casos de uso

En este diagrama:
- **Usuario (U)**: Representa a los usuarios del sistema, como los clientes que utilizan la aplicación.
- **Administrador (A)**: Representa a los administradores o personal interno que gestiona el sistema.

Los casos de uso incluidos son:
1. **Iniciar sesión**: Permite al usuario autenticarse en el sistema.
2. **Realizar búsqueda**: Permite al usuario buscar productos.
3. **Ver detalles del producto**: Permite al usuario ver información detallada sobre un producto.
4. **Agregar al carrito**: Permite al usuario agregar productos al carrito de compras.
5. **Realizar pago**: Permite al usuario completar la compra.
